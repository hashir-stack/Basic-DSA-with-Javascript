Indirect recursion in JavaScript happens when two or more functions call each other in a cycle, instead of a single function calling itself directly. It’s a mutual recursion pattern where the cycle continues until a base condition breaks the loop, preventing infinite calls.

🔹 Key Concept - Direct recursion: A function calls itself directly. - Indirect recursion: Function A calls Function B, and Function B calls Function A (or more functions in a chain). This pattern is useful when breaking down problems into smaller, alternating responsibilities between functions.
